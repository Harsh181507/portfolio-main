## 🛠️ Local Development

This project was bootstrapped with **Vite** and built using **React** and **Tailwind CSS**. The animations are powered by **Framer Motion**.

### Prerequisites
- Node.js (v16+) installed on your machine.

### Installation & Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Harsh181507/portfolio-main.git
   ```
2. Navigate to the `frontend` directory:
   ```bash
   cd portfolio-main/frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the local development server:
   ```bash
   npm run dev
   ```
